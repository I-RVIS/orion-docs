Orion Documentation
===================

Welcome to the official documentation of **Orion**.

Orion is a modular robotics framework designed for robotic arm control and autonomy.
It provides a complete pipeline from inverse kinematics solving to hardware control,
with built-in collision avoidance, real-time 3D visualization, and sensor integration.
